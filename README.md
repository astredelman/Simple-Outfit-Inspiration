# Simple Outfit Inspiration Website

## Project Overview

The Simple Outfit Inspiration Website is a beginner-friendly front-end web development project created using HTML, CSS, and JavaScript. The website provides users with outfit inspiration through a simple image gallery and includes basic interactive features.

The purpose of this project is to demonstrate fundamental front-end development skills, including creating multiple web pages, designing responsive layouts, styling with CSS, and adding interactivity with JavaScript.

---

## Target Audience

This website is designed for teenagers and young adults who are interested in fashion and outfit ideas. The simple layout allows users to easily browse different clothing styles.

---

## Website Features

- Home page introducing the website
- Outfit gallery page
- Filter buttons to display different outfit categories
- Contact page with a simple contact form
- Responsive design for desktop and mobile devices
- Navigation bar between pages
- Hover effects on buttons and images
- JavaScript confirmation message after submitting the contact form

## How to Run the Website

1. Download or clone the GitHub repository.
2. Open the project folder.
3. Double-click **index.html** or open it in your preferred web browser.
4. Use the navigation menu to explore the website.
